Privacy Policy

TTS Bot collects message content, server IDs, user IDs, command name + time of invoke, and any settings set using the /set command or by any other means.

TTS Bot needs this data to function correctly

  • Message content in the setup channel is required for basic function and caching. This is also shared with Google for TTS
  • Server IDs, user IDs, and settings are for storing in the database so TTS Bot can be configurable.
  • Command name + time of invoke are collected for analytics, so development can be focused in the right places.

If you have any concerns, please contact the development team via the support server (/invite) or contact Gnome!#6669 directly for data removal or security concerns.
